# java-cli-mill-h2-simple

## Description
A java millbuild build, that connects to h2
database.

H2 is a database written in `java`
and is best known for a `dev` only
use.

## Tech stack
- java
- millbuild
- h2 driver

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
